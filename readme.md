# WebRTC-network

Децентрализованная сеть, осуществляющая взимодействие через браузер. Для работы приложения необходимо устанавливить userscript, который входит в сеть через сигнальный сервер. На данный момент функционирует сигнальный сервер по адресу https://webrtc-classif-dhtnetwork.herokuapp.com

## UserScript

Для браузера Chrome необходима установка расширения Tampermonkey, для остальных браузеров использовать аналогичные способы подключения пользовательских скриптов.
